# Family Finance Manager 

A full-stack web application built with **Python** and **Flask** designed to manage and track family finances. It allows users to record income and expenses, categorize transactions, and manage multiple family members with role-based access control.

## Key Features
* **Role-Based Access Control (RBAC):** Different user roles (e.g., Parent, Child) with specific permissions and data visibility.
* **Transaction Tracking:** Add, edit, and categorize income and expenses per family member.
* **Secure Authentication:** Session-based user login, secure password handling, and form validation.
* **RESTful API Architecture:** Clean and modular HTTP API design for handling frontend requests.
* **Dynamic UI:** Server-side rendering using **Jinja2** templates.

## Technologies & Architecture
* **Backend:** Python, Flask
* **Architecture:** MVC-style modularization
* **Database:** Relational data model (Users, Roles, Transactions)
* **Frontend:** HTML5, CSS, Jinja2 Templating
* **Security:** Session management, secure user input handling
